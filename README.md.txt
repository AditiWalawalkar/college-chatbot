# College Chatbot – IBM Watsonx.ai Project

This project is a college information chatbot built using **IBM Watsonx.ai**.  
It uses vector indexes and text datasets to answer student-related queries.

## Project Structure
- `PBLProject_dataset.txt` – Text dataset for project-related queries  
- `unimate_chatbot_dataset.txt` – Chatbot training text data  
- `PBLProject_dataset` – Vector index asset for Watsonx.ai  
- `unimate_chatbot_dataset` – Vector index asset for Watsonx.ai  

## How It Works
The chatbot uses IBM Watsonx.ai to create embeddings and vector search through these datasets to generate context-aware responses.
